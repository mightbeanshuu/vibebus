ship it locally

Three commands.
No cloud bus.

Clone, link the CLI, then point any MCP-capable agent at vibebus-mcp.

install.sh 
$ git clone https://github.com/mightbeanshuu/vibebus.git
$ cd vibebus
$ npm link
mcp.json 
{
  "mcpServers": {
    "vibebus": {
      "command": "vibebus-mcp"
    }
  }
}

the protocol

How agents stay in sync.

01

Register

Each agent declares its id, CLI, role, model, workspace, and capabilities.

02

Handoff

Create a task and a required-ack message in one atomic local update.

03

Thread

Agents reply in context, read the full thread, and keep durable decisions separate.

04

Ack

Important work does not disappear into the void — recipients acknowledge receipt or completion.
